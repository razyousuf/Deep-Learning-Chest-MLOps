Brief Description

📌 About This Application

This web-based tool classifies chest CT scan images as either Healthy or showing signs of Coccidiosis. It uses deep learning techniques to assist in fast and reliable diagnosis.

🧪 How It Works

  • Image Upload: Users can upload a CT image for diagnosis.
  • Prediction: A trained CNN model based on VGG16 processes the image.
  • Result: The model outputs whether the image indicates Healthy or Coccidiosis.

🔬 Model Architecture

  • Base: Transfer learning with VGG16, pretrained on ImageNet.
  • Customization: Additional fully connected layers fine-tuned for binary classification.
  • Efficiency: Optimized for fast inference and minimal overfitting.

🛠️ Tech Stack

  • Frontend: Jinja2, HTML5, Bootstrap 4
  • Backend: Flask (Python)
  • Model: Keras with TensorFlow backend
  • Pipeline: Modular MLOps with endpoints for training and prediction

📊 Training Metrics

Once training is completed, the app automatically stores and displays evaluation metrics:

  • Accuracy and Loss are saved to scores.json
  • Displayed on the training results page for transparency

🧩 Model Training Endpoint

In addition to prediction, this app includes a dedicated training interface. Users can customize training parameters like:

  • Epochs, Batch Size, Learning Rate
  • Data Augmentation: Toggle on/off

The model is trained dynamically and saved with updated metrics.
